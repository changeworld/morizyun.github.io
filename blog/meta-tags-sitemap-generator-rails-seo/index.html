ヨヤクスリとは？

薬局、ユーザーも無料、お薬予約、薬局の口コミでポイントが付く

(1) 診察後、処方箋を受け取る (2) 薬局を選ぶ (3) 処方箋を撮影、送信する (4) 薬局でクスリをつけとる

調剤ポイントはNG

予約ポイントで10ポイント、口コミで5ポイント

38,000薬局で予約ができる。 薬局の口コミ、ファシリティ、外観等が1200件以上登録されている(どこから、誰が？) 処方履歴が、自動的に電子化される => ライフログ? じほうさんの時点 1.5万品目を見ることができる KC薬剤師?が質問解答をする

電子お薬手帳

  • 競合は20個くらいらしい？
  • 薬局と提携していないと読み込めない、QAコードがないと読み込めない
  • 共通化したプラットフォームがいい?

  • ジェネリックに変わった場合にクスリ情報が提供されない

  • 薬局側が激しく不満を持っている

  • データはビックデータとして勝手に使う

  • 薬局は上位10社で10数%になっている => いびつな業界

  • 中規模の薬局が積極的にやっている

  • コンシェルジュサービス => 患者さん向け、薬局向けの質問解答(電話解答)サービスを使う => SMS、メール電話

  • 海外では処方箋の回数券が一般的

電子処方箋の工程表

立ち上げのキッカケ

健康 x ネット x イノベーション x 処方衣料品 => ヨヤクスリを立ち上げた

おそらく運営コストは1千万-3千万くらい?

調剤薬工場を作って通販するのはいずれ普及する可能性がある

ユーザー構成比率

30-50代 が60%を使っている 男性の方が利用率は高い 70%以上が門前薬局以外で使っている(ちょっといびつ) 東京がよく使っている 処方日数が多い人が多い、処方点数が高い

ユーザーのニーズ

医薬分業 - 病院と薬局で二階話さないといけないのが面倒くさい

お医者さんの方が、薬局よりも強い

どこの病院 => 薬局かがわからない

どこの病院のどこの薬局かがわかる

薬局からお金を取ってくる(多分大きめの病院から大量に処方箋を取ってくる)

マネタイズのイメージ

薬局側に服薬指導を行う クスリのリマインダーを行う アドヒアランスサポート => AというクスリとBというクスリの飲み合わせの確認や、確認など ビックデータの活用
